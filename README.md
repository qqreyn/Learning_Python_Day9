# Learning_Python_Day9

Today's topic is variable scope!

**Goal:** Understand local vs global scope to avoid bugs and write cleaner functions

- **task1:** Create a function that modifies a local variable and observe the behavior

- **task2:** Use the `global` keyword to modify a global variable from inside a function

- **task3:** Write nested functions to explore different scope levels

- **task4:** Demonstrate variable shadowing using same-named variables

- **task5:** Debug code that contains scope-related errors

**mini-project:**
Build a small interactive program that demonstrates:

- Local vs global variables  
- Using the `global` keyword  
- Nested functions  
- Variable shadowing  
- Fixing/debugging scope errors
